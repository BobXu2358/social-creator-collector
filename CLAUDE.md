# Claude Code — Social Creator Collector

Follow `AGENTS.md` in this repo. Key points:

- This is a standalone Python toolkit — no OpenClaw required.
- Install deps: `python3 -m venv social/.venv && social/.venv/bin/pip install -r social/requirements.txt`
- Cookie onboarding: prefer local file (Mode A), accept chat paste (Mode B).
- Cookie-Editor extension: https://chromewebstore.google.com/detail/cookie-editor/hlkenndednhfkekhgcdicdfddnkalmdm
- Douyin per-video fan growth: DOM extraction from 投稿列表 tab (not API).
- Read `skills/social-creator-data/SKILL.md` for full workflows.
- Never print cookie/token values.
